Project Overview:

This project focuses on analyzing retail sales data using SQL and Python to generate key performance indicators (KPIs), identify trends, and build a visual sales analytics dashboard. The Superstore dataset is used to convert raw transactional data into meaningful business insights that support data-driven decision-making.


Objectives:

-> Store and manage sales data using SQL
-> Perform sales analysis using Python (Pandas)
-> Calculate key performance indicators (KPIs)
-> Identify trends and patterns in sales data
-> Visualize insights through charts and dashboards
